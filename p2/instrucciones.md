# **Instrucciones de ejecución**

1. El único archivo a ejecutar es **notebooks/experiment.ipynb**
2. Las **rutas** dentro del notebook son todas **relativas** para asegurar el correcto funcionamiento del código, por tanto, es necesario **mantener la estructura** de los directorios.
3. Es necesario crear el enviroment a partir de ../envP2/Project.toml para poder ejecutar el código correctamente.
4. El notebook se ha ejecutado con un kernel de **IJulia**, por eso no se ha incluído la creación del entorno en la primera celda del notebook.
5. En **src/modelFactory.jl**, se han comentado las líneas referentes a **Catboost** para evitar errores durante la ejecución (descomentar si se quiere probar, de todas formas, las demos cuentan con los imports necesarios).
6. Las demos de Catboost, están comentadas, ya que darán error.
